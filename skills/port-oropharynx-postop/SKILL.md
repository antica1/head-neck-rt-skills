## 术后 PORT 的独特挑战

### 术前 vs 术后——根本差异

| | 术前（根治性 RT） | 术后（PORT） |
|---|---|---|
| **靶区来源** | 影像学可见肿瘤 + 亚临床扩展 | **手术床 + 病理报告** |
| **解剖** | 正常解剖 | **解剖变形**（TORS术腔/缝合/水肿） |
| **核心文件** | MRI | **手术记录 + 病理报告** |
| **外放** | GTV + N mm | **手术床 + N mm（不同亚区不同）** |
| **关键风险** | 地理漏照 | **漏照 + 过度覆盖**（双方向） |

> **术后靶区勾画的黄金法则：手术记录和病理报告比 MRI 更重要。**

### 口咽癌 PORT 的特殊性

与口腔癌 PORT 的关键差异：

| | 口腔癌 PORT | 口咽癌 PORT |
|---|---|---|
| **RP** | 仅"后门"部位需要 | **✅ 必照** |
| **对侧颈** | 仅近中线或高危时 | 舌根/软腭→**双侧必需** |
| **手术方式** | 经口/经颈/下颌骨劈开 | 多为 **TORS**（经口机器人） |
| **皮瓣** | 前臂/腓骨/ALT | 较少需要大皮瓣 |
| **HPV** | 多为 HPV(−) | 多为 HPV(+)→预后较好 |

## 二、口咽癌——按亚部位精确定义

### 2.0 亚部位清单

| 部位 | 亚部位 | 关键差异 |
|------|--------|---------|
| **扁桃体** | 向上（软腭）/向下外侧（舌根） | 扩展方向决定 CTV |
| **舌根** | 中线 vs 偏侧 | 中线→双侧 II-IV+RP |
| **软腭** | 口腔面 vs 鼻咽面 | 鼻咽面→咽鼓管开口 |
| **咽侧壁** | 口咽侧壁 / 下咽侧壁 | 下咽→增加 VI 区 |

### 2.1 扁桃体癌

**向上扩展（软腭方向）**：咽侧壁+软腭切除缘+1.5cm，同侧软腭至中线，腭帆提肌/张肌残余全纳入。

**向下外侧扩展（舌根方向）**：咽侧壁+同侧舌根+1.5-2cm，舌根后1/3纳入，CN IX PNI(+)→颈静脉孔。

**共同要求**：咽旁间隙（TORS 深部切除→同侧），翼下颌缝（深及→纳入），颈部：同侧 II+III+IV+RP。

### 2.2 咽侧壁癌

| CTV 组分 | 范围 |
|---------|------|
| 手术床 | 咽侧壁术腔+1.5-2cm |
| 上界 | 软腭水平（口咽）或舌骨水平（下咽） |
| 下界 | 会厌谷（口咽）或梨状窝外侧壁（下咽） |
| 后界 | 咽后壁黏膜 |
| 颈部 | 口咽→同侧 II+III+IV+RP；下咽→同侧 II+III+IV+**VI区**+RP |

> 下咽侧壁→增加 VI 区。口咽侧壁不照 VI。

### 2.3 软腭癌

| 面 | CTV 要求 |
|----|---------|
| 口腔面（下方） | 术腔+1-1.5cm |
| 鼻咽面（上方） | 若切除达鼻咽面→同侧咽鼓管圆枕+咽隐窝纳入 |

### 2.4 舌根癌

| 位置 | CTV 颈部 |
|------|---------|
| 偏侧 | 同侧 II+III+IV+RP |
| 中线/双侧 | **双侧 II+III+IV+双侧 RP** |
| 近会厌谷 | 向下至会厌谷和舌会厌皱襞 |

## 四、切缘 + PNI + 剂量（与口腔癌 PORT 一致）

**切缘**：≥5mm→0mm 外放/50-54Gy；1-4mm→+3-5mm/56-60Gy；R1→+5-10mm/60-66Gy；R2→+10-15mm/66-70Gy。

**PNI**：镜下→沿神经至主要分支；知名神经→颅底孔道（CN IX→颈静脉孔，V3→卵圆孔）。

**剂量**：术后不使用 SIB。PTV-66-70（R1/R2/ENE+）、PTV-60（手术床）、PTV-54（预防区）、PTV-50（低风险）。

## 附：九院口咽癌 198 例复发分析关键教训

| 病例 | 教训 |
|------|------|
| 软腭+咽侧壁，术后 PORT | 🔴 单侧颈清+中线跨过→对侧颈复发。验证"单侧颈清→对侧开放"规则 |
| 舌根 HPV+，根治性 RT | ⚠️ 舌根 HPV+ 即便早期也需双侧颈覆盖 |
| 舌根 ENE+，术后 PORT | ⚠️ ENE+ 术后仍需注意原发灶床充分覆盖 |

> **核心教训**：单侧颈清+口咽中线/近中线原发灶→对侧颈复发风险真实存在。
